# Repository for Miyashita et al. (2026)

This repository contains the R scripts and source data used to analyze the data and to generate the figures for the manuscript:
** "Diel and Seasonal Variation in Surface Water CO2 Partial Pressure in an Agricultural Reservoir in Japan"**
submitted to *Journal of Geophysical Research: Biogeosciences*.

## Files

- **JGR_MIYASHITAetal2026_Rscript.R**  
  R script for generating Figures 2–8 and S4–7, and for performing statistical analyses of these figures and Table 2–3 and S1

- **MIYASHITAetal2026_FigS2.xlsx**  
  Data and calculations used to generate Figure S2

- **MIYASHITAetal2026_FigS3.xlsx**  
  Data and calculations used to generate Figure S3

- **MIYASHITAetal2026_FigS8.xlsx**  
  Data used to generate Figure S8 (using SigmaPlot)
